Sposoby badania rozkładu zmiennych

Sposoby zależą od typów zmiennych. Prawidłowe określenie na wstępie typów zmiennych pozwala na bezbłędny wybór metody/wizualizacji do analizy rozkładu.

Dane

Dane o pingwinach, dostępne w pakiecie palmerpenguins. Zbiór danych jest podobny do popularnego zbioru danych iris.

Zmienne (kolumny)

  • species: gatunek (Adélie, Chinstrap, Gentoo)
  • island: wyspa (Biscoe, Dream, Torgersen)
  • bill_length_mm / bill_depth_mm: wymiary dzioba
  • flipper_length_mm: długość skrzydła
  • body_mass_g: masa ciała
  • sex: płeć
  • year: rok

Typy zmiennych

  • zmienne jakościowe
  • zmienne ilościowe

Określamy typy zmiennych

  • species: jakościowa, nominalna
  • island: jakościowa, nominalna
  • bill_length_mm / bill_depth_mm: ilościowa, ilorazowa
  • flipper_length_mm: ilościowa, ilorazowa
  • body_mass_g: ilościowa, ilorazowa
  • sex: jakościowa, binarna
  • year: ilościowa, przedziałowa

Rozkład zmiennej species - barplot()

Wykres słupkowy z wykorzystaniem funkcji barplot().

! Przed użyciem funkcji barplot() należy zliczyć obserwacje.

barplot(table(penguins$species),
        main = "Liczność gatunków")

Rozkład zmiennej species - geom_bar()

Wykres słupkowy z wykorzystaniem funkcji geom_bar()

! geom_bar() - zlicza występowanie kategorii

ggplot(data = penguins,
       mapping = aes(x = species)) + 
  geom_bar(color = "#ea71ea", fill = "purple") +
  labs(title = "Rozkład zmiennej species ", 
       x = "",
       y = "Liczność") +
  theme_minimal()

Rozkład zmiennej species - geom_col()

Wykres słupkowy z wykorzystaniem funkcji geom_col()

! geom_col() - podajemy w x/y kategorię i odpowiednio w x/y ich liczność

df <- data.frame(table(penguins$species))
names(df) <- c("Gatunek", "Count")

ggplot(df,
       aes(x = Gatunek ,
           y = Count)) + 
  geom_col(color = "blue", fill = "white") +
  labs(title = "Rozkład zmiennej species", 
       x = "",
       y = "Liczność") +
  theme_bw()

Rozkład zmiennej species - wykres kołowy

W pakiecie ggplot2 nie istnieje funkcja do rysowania wykresu kołowego. Aby taki wykres uzyskać należy przygotować wykres słupkowy i zmienić współrzędne na biegunowe.

df <- data.frame(table(penguins$species))
names(df) <- c("Gatunek", "Count")

ggplot(df,
       aes(x = "",
           y = Count,
           fill = Gatunek)) + 
  geom_col(weight = 1) +
  labs(title = "Rozkład zmiennej species", 
       x = "",
       y = "Liczność") +
  coord_polar("y") +
  theme_void()

Wykres punktowy - base R

plot(penguins$body_mass_g, penguins$flipper_length_mm,

     xlab = "Masa [g]",
     ylab = "Długość skrzydła [mm]")

Wykres punktowy - ggplot2

ggplot(penguins, aes(x = body_mass_g, y = flipper_length_mm,
                     shape = species)) +
  geom_point() +
  labs(title = "Zależnośc masy od długości skrzydła",
     x = "Masa [g]",
     y = "Długość skrzydła [mm]",
     shape = "Gatunek") +
  theme(legend.position = "top")

Koleracja

Domyślnie jest używana korelacja Pearsona.

\(r_{xy} = \frac{\sum_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^{n} (x_i - \bar{x})^2 \sum_{i=1}^{n} (y_i - \bar{y})^2}}\)

Można wyliczać również współczynnik korelacji Spearmana, który bada monotoniczność.

df <-  na.omit(penguins) # usuwamy braki danych
cat("Współczynnik Pearsona")
Współczynnik Pearsona
cor(df$body_mass_g, df$flipper_length_mm)
[1] 0.8729789
cat("Współczynnik Spearmana")
Współczynnik Spearmana
cor(df$body_mass_g, df$flipper_length_mm, method = c("spearman"))
[1] 0.8403902
